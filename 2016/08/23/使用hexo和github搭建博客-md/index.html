使用hexo和github搭建博客

2016.08.23 Weapon-x  热度 ℃ 条评论

前言

现在工程师出门找工作，github和博客几乎成了标配或者是加分项，之前在博客园和其他的一些博客网站上记录一些自己的学习过程，总觉得还是不够爽，还是转战github pages+hexo，上面提供了300M的免费空间，还是非常靠谱的。Hexo是一个简单、快速、强大的博客发布工具，我们可以使用markdown语法进行写作。

正片

操作系统：OSX，如果是linux或是windows的需要自行兼容一下

安装依赖

安装hexo之前我们需要安装好git、node、npm(node package manager)

如果已经装好如上图所示，那么再安装hexo到全局下:

npm intall -g hexo

安装完输入 hexo -v 看看是否安装成功，如果有版本信息的输出就说明安装成功啦

新建仓库

因为博客是托管在github上的，如果没有账号的话需要先注册一个，然后新建一个仓库

注意仓库的名字是:username.github.io，我的username就是weapon-xx

然后回到本地文件夹下新建一个任意名字的文件夹，我这里就叫做weapon-xx吧，然后我们从终端里进入到该文件夹下输入命令hexo init，初始化hexo项目，当初始化完成后会有一堆输出信息，我们也可以看到该文件夹下也多出了hexo相关文件

在终端下再输入以下命令:

  • hexo generate
  • hexo server

可以看到hexo项目已经在4000端口运行了，在浏览器中输入url，可以看到如下效果，说明成功一大半啦

主题配置

这个也非常简单，我使用的是hexo-theme-next主题，在本文发表前有4847个⭐️，传送门 hexo-theme-next
然后打开终端，在项目目录下面输入命令

git clone https://github.com/iissnan/hexo-theme-next themes/next

等安装好themes文件夹下就会多出一个next文件夹，在项目配置文件 _config.yml 中找到themes字段，把它的值改成next就可以了，在重启下服务器看看效果吧

next主题的支持三种外观，五中代码高亮，主题风格简单又不失优雅，文档地址是NexT，大家可以进一步了解主题的配置

还有一些在github上⭐️较多的主题，大家可以挑个自己喜欢的：

部署

现在我们需要在项目的配置文件 _config.yml 里配置一下项目的github地址，拉到最后一行会发现deploy的关键字，type和branch都是一样的，repository就是自己的项目地址

再打开终端输入以下命令

npm install hexo-deployer-git –save
hexo clean
hexo generate
hexo deploy

第一次部署的时候需要填写github的用户名和密码，如果不出意外的话，可以在github仓库看到原本是空的仓库已经多了一些文件，然后再到浏览器打开项目地址，http://用户名.github.io/ 就可以看到效果

总结

折腾了一下午总算把博客搭建完成了，总的上来说没有太大的困难，但是需要一点耐心，在日后博客的一些功能完善、第三方配置会另外写一篇博文来记录，如果在搭建过程中遇到什么问题或者文中出现错误的地方，欢迎留言
