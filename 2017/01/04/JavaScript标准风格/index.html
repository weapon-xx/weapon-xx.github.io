JavaScript标准风格

2017.01.04 Weapon-x  热度 ℃ 条评论

学习一种风格最好的方式就是将它应用在你的代码上。

规则

  • 使用两个空格缩进
  • 使用单引号避免转义
  • 避免出现没使用的变量
  • 在关键字后面添加空格
  • 在函数声明括号前添加空格
  • 使用 === 代替 ==
  • 运算符必须间隔
  • 逗号后面 必须有空格
  • 让else语句和大括号在同一行
  • 对于多行的if语句，请使用花括号
  • 始终处理函数参数错误
  • 始终使用浏览器全局变量作为前缀
  • 避免多行空行
  • 对于多行的三元操作符，将? 和 : 放在单独一行
  • 对于var声明，将每个生命写在单独的语句
  • 使用额外括号包裹条件语句

分号

  • 去掉分号
  • 不要以(, [, ` 作为一行的开始。这是省略分号的唯一办法，这样可以避免一些潜在的问题

总结

其实在工作中，大部分时间我们都在阅读代码然后再开始写代码，当看到乱七八糟的代码的时候肯定都不想看了。所以，从一开始养成好的代码习惯，方便自己，也方便他人的阅读：）
